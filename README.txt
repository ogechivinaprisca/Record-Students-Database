THEORY QUESTIONS
QUESTION 1:
Explain the difference between a view function and pure function.
Answer:
View function declares that no state will be changed. 
Pure function declares that no state variable will be changed or read.

QUESTION 2:
Explain the difference between a payable and non-payable function.
Answer: 
A Payable functions is a function that can receive ether.
On the other hand, non-paybale function cannot receive ether.

Question 3:
Explain the difference between a constant data type and immutable data type.
For constant data type, the value has to be fixed at compile-time, 
while for immutable , it can still be assigned at construction time.
In both cases, the variables cannot be modified after the contract has been constructed

QUESTION 4:
Explain the difference between a Read Function and Write function.
Answer: 
A Read function can only view the state variable & cannot change it.
On the othe hand, a write function can view & make changes to the state variable.
